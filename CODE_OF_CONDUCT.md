# 贡献者行为准则

## 承诺

为了营造一个开放友好的环境，我们承诺以专业和尊重的态度对待所有参与者和贡献。

## 标准

鼓励的行为：

- 展示对他人的同理心和善意
- 尊重不同观点和经验
- 给予建设性反馈，优雅地接受批评
- 关注对社区最有利的事情

不可接受的行为：

- 使用色情化语言或 imagery、性关注或挑逗
- 人身攻击、辱骂、贬损评论
- 公开或私下骚扰他人
- 未经明确许可泄露他人私人信息

## 执行

违反准则的行为可向仓库维护者报告。维护者有权删除、编辑或拒绝不符合准则的评论、提交、代码、wiki 编辑、issue 和贡献。

## 适用范围

本准则适用于所有项目空间，以及在公共空间中个人代表项目时的行为。

## 来源

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) 2.1 版。
